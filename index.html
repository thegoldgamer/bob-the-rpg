About

This page is a simple wrapper to host a Gamemaker (HTML5) export. It loads /game/index.html in an iframe kept at the game's original dimensions and centered at the top of the page. You can add more content below and it will be scrollable.

Quick checklist

  • Export your game from GameMaker as HTML5.
  • Copy the entire exported folder contents into the repository's game/ folder (so the exported index.html ends up at /game/index.html).
  • If the game's resolution looks wrong, edit --game-width and --game-height variables at the top of index.html to match the game's original canvas size.

When you're ready to publish, you can enable GitHub Pages for this repository (set source to the main branch) and the game will be available at https://thegoldgamer.github.io/bob-the-rpg/.
